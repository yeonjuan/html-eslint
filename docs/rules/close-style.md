# close-style

This rule enforces a consistent closing style for HTML elements — either self-closing (`<br />`) or explicit close tags (`<br>`).

## How to use

```js,.eslintrc.js
module.exports = {
  rules: {
    "@html-eslint/close-style": "error",
  },
};
```
