## Disallow inline styles.

### Rule Details

This rule disallow the use of inline styles.

Examples of **incorrect** code for this rule:

```html
<div style="color:#ff0a00"></div>
```

Examples of **correct** code for this rule:

```html
<div class="some-color"></div>
```
