@html-eslint

An ESLint plugin to lint and fix HTML code.

  • Getting Started
  • Playground

What does @html-eslint do?

ESLint is a popular linting tool, but it only supports JavaScript.
This plugin allows you to lint your HTML code without having to install a separate linting tool or worry about editor support.

What rules are provided?

This plugin provides rules for Best Practices , SEO , Accessibility and Style .
If you can't find the rule you're looking for, please open an issue here.
