# `<html>` 태그에 `lang` 속성 사용하기

HTML 의 `lang` 속성은 웹 컨텐츠에서 사용하는 언어를 표시합니다.
`lang` 속성을 통해 검색 엔진에게 컨텐츠 언어에 대한 정보를 제공합니다.

## 규칙 설명

이 규칙은 `<html>` 태그에 `lang` 속성을 사용하도록 합니다.

**올바르지 않은** 코드 예시:

```html
<html>
  ...
</html>

<html lang="unkown">
  ...
</html>
```

**올바른** 코드 예시:

```html
<html lang="ko">
  ...
</html>
```
